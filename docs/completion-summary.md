## 技术亮点

### 1. 完整的双端联动架构
- 家长端（Web + 小程序）与学生端（React Native + Web）实时同步
- WebSocket实现毫秒级实时通信
-线数据缓存与自动同步

### 2. 智能辅助功能
- 拼音标注降低低龄学生阅读门槛
- 本地语音合成解放双眼
- 智能提醒防止作业遗漏

### 3. 科学的学习管理
- 番茄钟专注模式培养专注力
- 积分奖惩体系激励学习
- 数据统计提供科学反馈

### 4. 家长友好设计
- 远程监控管理，降低监督成本
- 数据可视化，直观了解学习情况
- 操作简单，零学习门槛

### 5. 学生端极简设计
- 纯学习向界面，无娱乐功能干扰
- 操作简单，低龄学生可独立使用
- 专注模式强制锁机

- Node.js >= 16.0.0
- PostgreSQL >= 13.0
- React Native CLI

1. 数据库初始化（执行database/init.sql）
2. 后端服务部署（配置环境变量、启动服务）
3. 学生端部署（React Native打包、Web构建）
4. 家长端部署（Web构建、小程序上传）
5. Nginx配置（反向代理、SSL证书）
