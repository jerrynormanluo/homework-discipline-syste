## 系统概述

青少年作业自律辅助系统是一款面向小学至初中学生的家庭教育辅助工具，通过家长管理端和学生显示端的联动，帮助培养孩子的作业自律性，降低家长监督成本。

### 主要功能

- **作业管理**: 远程发布、编辑、删除作业
- **智能提醒**: 作业开始、截止、超时提醒
- **专注模式**: 番茄钟专注学习，强制锁机
- **拼音辅助**: 低龄学生拼音标注功能
- **语音播报**: 作业内容语音朗读
- **积分奖惩**: 学习积分体系，奖励兑换
- **数据统计**: 学习数据可视化分析

## 家长端使用指南

### 1. 注册与登录

#### 注册账号
1. 打开家长端应用或访问网页
2. 点击"注册"按钮
3. 输入手机号、昵称、密码
4. 选择角色为"家长"
5. 完成注册

#### 登录系统
1. 输入手机号和密码
2. 点击"登录"按钮
3. 首次登录需绑定学生设备

### 2. 绑定学生设备

#### 添加学生
1. 进入"系统设置"页面
2. 点击"添加学生"按钮
3. 输入学生手机号
4. 设置设备名称和类型
5. 可选设置锁屏密码和音量限制
6. 点击"绑定"完成

#### 设备类型说明
- **安卓平板**: 适合学生日常使用
- **Windows电脑**: 适合家庭作业场景
- **智能显示器**: 适合固定位置学习
- **安卓电视**: 适合大屏展示

### 3. 作业管理

#### 创建作业
1. 进入"作业管理"页面
2. 点击"创建作业"按钮
3. 填写作业信息：
   - 选择学生
   - 输入作业标题
   - 选择学科和分类
   - 填写作业内容
   - 设置预计耗时
   - 选择截止时间和优先级
4. 点击"创建"完成

#### 编辑作业
1. 在作业列表中找到目标作业
2. 点击"编辑"按钮
3. 修改作业信息
4. 点击"更新"保存

#### 删除作业
1. 在作业列表中找到目标作业
2. 点击"删除"按钮
3. 确认删除操作

#### 作业分类说明
- **校内作业**: 学校布置的日常作业
- **加餐作业**: 额外的练习作业
- **背诵任务**: 需要背诵的内容
- **错题整理**: 错题复习和整理
- **课外阅读**: 课外阅读任务

### 4. 学生监控

#### 查看学生概览
1. 在首页点击学生卡片
2. 查看学生详细信息
3. 查看作业完成情况
4. 查看专注记录
5. 查看积分变动

#### 强制专注模式
1. 进入学生详情页面
2. 点击"强制专注"按钮
3. 设置专注时长
4. 点击"下发专注模式"
5. 学生端将自动进入专注状态

### 5. 积分管理

#### 设置积分规则
1. 进入学生设置页面
2. 切换到"积分规则"标签
3. 查看现有规则
4. 可通过后端API添加自定义规则

#### 创建奖励
1. 进入学生设置页面
2. 切换到"奖励设置"标签
3. 设置奖励名称和所需积分
4. 选择奖励类型
5. 学生可使用积分兑换

#### 奖励类型说明
- **娱乐时长**: 游戏或娱乐时间
- **户外活动**: 户外运动或游玩
- **小礼品**: 实物奖励
- **免加餐作业**: 免除一次加餐作业

### 6. 数据统计

#### 查看学习统计
1. 进入"数据统计"页面
2. 选择学生和时间范围
3. 查看作业完成趋势
4. 查看专注时长趋势
5. 查看积分变动趋势
6. 查看学科学习分布

#### 导出报表
1. 在统计页面点击"导出"
2. 选择导出格式（Excel/PDF）
3. 下载报表文件

### 7. 系统设置

#### 专注模式设置
1. 进入学生设置页面
2. 切换到"专注设置"标签
3. 配置专注时长和休息时长
4. 启用/禁用强制锁机模式
5. 选择专注模式类型

#### 设备配置
1. 修改锁屏密码
2. 调整最大音量限制
3. 设置静默时间段
4. 解绑学生设备

## 学生端使用指南

### 1. 注册与登录

#### 注册账号
1. 打开学生端应用
2. 点击"注册"按钮
3. 输入手机号、昵称、密码
4. 选择角色为"学生"
5. 完成注册

#### 登录系统
1. 输入手机号和密码
2. 点击"登录"按钮
3. 进入作业主页

### 2. 作业主页

#### 查看作业列表
- 作业按优先级和截止时间排序
- 超时作业标红显示
- 显示作业状态和剩余时间

#### 作业状态说明
- **未开始**: 作业尚未开始
- **进行中**: 正在做作业
- **已完成**: 作业已完成
- **已超时**: 超过截止时间

#### 开始作业
1. 点击作业卡片进入详情
2. 点击"开始作业"按钮
3. 状态变为"进行中"

#### 完成作业
1. 完成作业后点击"完成"按钮
2. 状态变为"已完成"
3. 等待家长核验

### 3. 作业详情

#### 查看作业信息
- 作业标题和学科
- 作业内容和附件
- 截止时间和预计耗时
- 实际完成耗时

#### 使用拼音功能
1. 点击"显示拼音"按钮
2. 汉字上方显示拼音标注
3. 再次点击关闭拼音

#### 播放语音讲解
1. 如果有语音讲解附件
2. 点击"播放语音讲解"按钮
3. 系统朗读作业内容

### 4. 专注模式

#### 进入专注模式
1. 在作业详情页点击"专注模式"
2. 选择专注时长（15/25/30/45分钟）
3. 点击"开始专注"
4. 进入专注学习状态

#### 专注模式特点
- 屏幕锁定，无法切换应用
- 显示倒计时
- 专注结束后自动休息
- 休息时间结束后可继续或退出

#### 退出专注模式
1. 点击"结束专注"按钮
2. 确认退出操作
3. 记录专注时长

### 5. 积分中心

#### 查看积分余额
- 首页显示当前积分
- 积分中心查看详细余额

#### 查看积分记录
- 进入积分中心
- 切换到"积分记录"标签
- 查看积分获得和扣除记录

#### 兑换奖励
1. 进入积分中心
2. 切换到"奖励兑换"标签
3. 选择想要兑换的奖励
4. 确认兑换
5. 等待家长审核

### 6. 我的页面

#### 查看个人信息
- 昵称和头像
- 绑定的家长信息
- 设备信息

#### 查看成长勋章
- 查看已获得的勋章
- 查看勋章解锁条件

#### 查看学习小结
- 今日作业完成情况
- 学习优点和不足
- 改进建议

## 常见问题

### 家长端常见问题

#### Q: 如何添加多个学生？
A: 在系统设置中可以添加最多3个学生设备。

#### Q: 学生端无法接收作业？
A: 检查网络连接，确认学生端已登录且设备正常在线。

#### Q: 如何修改作业截止时间？
A: 在作业管理页面找到作业，点击"编辑"修改截止时间。

#### Q: 积分规则如何设置？
A: 积分规则需要在后端通过API配置，前端只能查看现有规则。

### 学生端常见问题

#### Q: 忘记密码怎么办？
A: 联系家长重置密码，或使用注册手机号找回。

#### Q: 拼音功能不显示？
A: 检查拼音设置，确认功能已启用。

#### Q: 语音播报没有声音？
A: 检查设备音量设置，确认语音功能已启用。

#### Q: 专注模式无法退出？
A: 专注模式下需要等待专注时间结束，或联系家长强制结束。

#### Q: 积分兑换后多久生效？
A: 兑换后需要等待家长审核，审核通过后生效。

## 功能说明

### 拼音标注功能

#### 功能介绍
为作业内容中的汉字标注拼音，帮助低龄学生理解作业要求。

#### 使用方法
1. 在作业详情页点击"显示拼音"
2. 汉字上方显示标准拼音
3. 支持过滤高频简单汉字

#### 适用场景
- 语文作业
- 背诵任务
- 课外阅读
- 其他文字类学习任务

### 语音播报功能

#### 功能介绍
将作业内容转换为语音朗读，解放双眼，辅助学习。

#### 播报类型
- 新作业自动播报
- 作业内容手动朗读
- 提醒通知语音播报
- 休息时间语音提醒

#### 使用方法
1. 在作业详情页点击播放按钮
2. 系统朗读作业内容
3. 支持暂停和重播

### 专注模式功能

#### 功能介绍
基于番茄工作法的专注学习模式，帮助提高学习效率。

#### 模式类型
- **番茄钟**: 25分钟专注 + 5分钟休息
- **长时段刷题**: 45分钟专注 + 15分钟休息
- **自定义**: 自定义专注和休息时长

#### 强制锁机
- 开启后学生端无法切换应用
- 只能查看当前作业
- 有效防止分心

### 积分奖惩体系

#### 积分获得
- 按时完成作业
- 专注学习
- 连续打卡
- 完成额外任务

#### 积分扣除
- 作业超时
- 中途退出专注
- 拒绝完成任务
- 其他违规行为

#### 奖励兑换
- 使用积分兑换奖励
- 需要家长审核
- 审核通过后生效

### 数据统计功能

#### 统计维度
- 作业完成率
- 超时作业数
- 平均完成耗时
- 累计专注时长
- 各学科学习占比

#### 统计周期
- 日统计
- 周统计
- 月统计
- 自定义时间段

#### 可视化展示
- 趋势图表
- 饼图分布
- 数据表格
- 导出报表
